Grey-box models

Problem : "As an engineer, my experiments give me many values (not just the main one), can we use them all to improve my model ?"

Answer : Yes we can !

As long as we can compute the covariance between 2 points, regardless of their level's appartenance (to the main value or an other one).

Grey-box models: co-kriging

Example: derivative

c(ddxf(x1),f(x2))=ddxc(f(x1),f(x2))c(ddxf(x1),f(x2))=ddxc(f(x1),f(x2))

Grey-box models: co-kriging

Example: multi-fidelity

f~(x)=ρf(x)+bias(x)f~(x)=ρf(x)+bias(x)
The biasbias is supposed uncorrelated with ff.
